== Description ==

**WooCommerce Category Block** is a powerful Gutenberg block plugin that allows you to display WooCommerce product categories in stunning **grid** or **slider** layouts with advanced filtering options.

Perfect for WooCommerce stores that want to improve product discovery, enhance navigation, and showcase categories beautifully on any page.

== Installation ==

### Automatic Installation

1. Go to WordPress Admin ‚Üí Plugins ‚Üí Add New
2. Search for **"WooCommerce Category Block"**
3. Click **Install Now**
4. Click **Activate**
5. Done! Start using the block

### Manual Installation

1. Download the plugin ZIP from [GitHub](https://github.com/jenishwpsd-afk/wc-category-block)
2. Go to WordPress Admin ‚Üí Plugins ‚Üí Add New ‚Üí Upload Plugin
3. Choose the downloaded ZIP file
4. Click Install Now and then Activate
5. The block is ready to use

### Requirements

* WordPress 6.0 or higher
* WooCommerce 6.0 or higher
* PHP 7.4 or higher

### After Installation

1. Make sure WooCommerce is installed and activated
2. Create some product categories in WooCommerce
3. Go to any page/post editor
4. Add the "WC Category Block" from the block inserter

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes, WooCommerce must be installed and activated for this plugin to work.

= Can I display categories in a slider? =

Yes! The plugin supports both grid and slider layouts. Choose your preferred layout from the block settings.

= How many columns can I display? =

You can choose between 1-6 columns for both grid and slider layouts.

= Can I filter specific categories? =

Yes, you can either show all categories or limit to a specific number (1-50 categories).

= Can I sort categories? =

Yes! Sort by name, product count, category ID, or slug in ascending or descending order.

= Does it show product counts? =

Yes, you can toggle the display of product counts for each category.

= Is it mobile responsive? =

Absolutely! The block is fully responsive and optimized for mobile, tablet, and desktop devices.

= Can I hide empty categories? =

Yes, there's an option to hide categories that have no products.

= Can I customize the styling? =

Yes! The plugin provides CSS classes that you can use to customize the appearance. Add custom CSS in your theme or via the Customizer.

= Does it work with any WordPress theme? =

Yes, it works with any theme that supports Gutenberg blocks.

= Is it translation ready? =

Yes, the plugin is fully internationalized and ready for translation.

= Does it affect site performance? =

No, the plugin is lightweight and optimized for performance with lazy loading and efficient queries.

= Can I use multiple blocks on the same page? =

Yes, you can add as many category blocks as you need on any page.

= Does the slider work on touch devices? =

Yes, the slider is touch-enabled and works perfectly on mobile and tablet devices.

= Can I change the slider speed? =

Currently, the slider uses default settings. Custom speed options may be added in future updates.

= How do I report a bug? =

Please report bugs on our [GitHub Issues page](https://github.com/jenishwpsd-afk/wc-category-block/issues).

= How can I contribute? =

We welcome contributions! Visit our [GitHub repository](https://github.com/jenishwpsd-afk/wc-category-block) to contribute.
