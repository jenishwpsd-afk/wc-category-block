Display WooCommerce product categories in a beautiful grid or slider layout with filtering support using a Gutenberg block.

== Description ==

WC Category Block is a powerful Gutenberg block plugin that allows you to showcase WooCommerce product categories in a modern and user-friendly way.

It helps improve store navigation by allowing users to filter and browse categories easily.

Perfect for WooCommerce stores that want better UI and improved product discovery.

== Features ==

- Display WooCommerce product categories
- Grid and slider layout options
- Category filter functionality
- Gutenberg block integration
- Fully responsive design
- Lightweight and fast
- Easy customization

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wc-category-block` directory, or install via the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Make sure WooCommerce is installed and activated.
